SERVER
Per stampare le informazioni (url, method, time, ...) relative al server usare
echo '<pre>';
var_dump($_SERVER);
echo '</pre>';
exit;

Per ottenere il valore di un singolo attributo del server
es: $_SERVER['REQUEST_METHOD']
